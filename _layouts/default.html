Martes 21 -Miercoles 22 de Febrero
Open Data Geo Meetup Lima

Realizar mejores aplicaciones con Datos Abiertos y Tilemill Gratis

Participa y aprenda a desarrollar mapas personalizadas y a medida con Tilemill, con el cual aprenderá paso a paso desde cero la manera de diseñar mapas con Tilemill sin ser un especialista en cartografía, El cual tiene por objetivo hacer uso de datos abiertos como principal fuente de datos para desarrollar mejores aplicaciones y muchas otras alternativas que crea conveniente desarrollar con Tilemill para el mejor desarrollo de su ciudad, pais etc..

Comentarios

Aprenda a diseñar mapas en Tilemill de 5pm-7pm

1. Fundamentos teóricos e Instalación de Tilemill

Ian Ward 5:00pm-5:30pm

Se Realizara introducción a Tilemill, como un programa de software libre para la elaboración de Cartografía digital, con la cual se pueden diseñar mapas, personalizadas y a medida de cada uno.

    Los puntos a tratar:
  • Tilemill
  • Mapnik
  • Carto
  • Node.js
  • Mapbox Hoting
  • Instalación de Tilemill

2. Fuentes y preparación de datos para la utilización con Tilemill

Ruben Lopez de 5:30pm 6:30pm

En esta Sección se mencionaran las principales fuentes de obtención de datos Geográficos y como es el procesamiento de estos datos para llegar a realizar mapas personalizados en Tilemill

    Los puntos a tratar:
  • Archivos .DWG(archivos autocad)
  • Fuente OSM
  • Carga de datos OSM a base de datos Postgis
  • Unión de datos geográficos y datos que usted crea conveniente mostrar geográficamente

3. Diseño de Mapas con Tilemill.

(Ian o Ruben) de 6:30pm-7:00pm

En este Sección diseñaremos mapas personalizado con Tilemill, mediante código básico css de lado del servidor, en los cuales aprenderá un panorama de como desarrollar sus propios mapas.

    Los puntos a tratar:
  • Importación de archivos shp a Tilemill.
  • Importacion de datos de base de datos Postgis.
  • Diseño de estilos en Tilemill.
  • Exportacion de mapas para la web con Tilemill.
  • Subiendo mapas a Mapbox Hosting gratuitamente.

Diseñe su propios mapas con Tilemill de 7pm-9pm


Tendrá un tiempo de 2 hora para desarrollar sus propios mapas personalizados con Tilemill, en el cual sera asesorado por Ian ward y Ruben Lopez, quienes podrán aclara sus dudas en la preparación de datos geográficos para el uso en Tilemill y en el Diseño de mapas en Tilemill.
